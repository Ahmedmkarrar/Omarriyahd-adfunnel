# Property Photos

Add property photos from the Redfin listing to this directory.

## Required Photos

1. `hero-bg.jpg` - Main hero background (exterior drone shot or front view)
2. `exterior-front.jpg` - Front of house
3. `exterior-side.jpg` - Side view with landscaping
4. `living-room.jpg` - Main living area
5. `kitchen.jpg` - Kitchen overview
6. `kitchen-island.jpg` - Kitchen island detail
7. `primary-bedroom.jpg` - Primary bedroom
8. `primary-bath.jpg` - Primary bathroom
9. `view.jpg` - View from property
10. `patio.jpg` - Outdoor patio/deck
11. `backyard.jpg` - Backyard/grounds
12. `garage.jpg` - Garage exterior
13. `aerial.jpg` - Aerial/drone view

## Recommended Specifications

- Minimum resolution: 1920x1280 pixels
- Format: JPG (optimized for web)
- File size: Under 500KB each (compress using TinyPNG or similar)

## Agent Photo

Add agent photo as:
- `../agent-photo.jpg` (in the public folder root)
- Recommended: Professional headshot, 400x400 pixels minimum

## OG Image

Create an Open Graph image for social sharing:
- `../og-image.jpg` (in the public folder root)
- Dimensions: 1200x630 pixels
- Include property exterior + price + address
